**Covers**

- Intervalo de confianza en diferencia de medias excluye 0 → rechazar H0
- El valor crítico que determina rechazo es exactamente 0 (no otro número)
- Al rechazar H0 se 'acepta' la hipótesis alternativa

**Front**

En el contraste de hipótesis de una diferencia de medias, ¿qué conclusión se extrae cuando el intervalo de confianza excluye el valor 0?

**Back**

Se rechaza la hipótesis nula (H0) y se 'acepta' la hipótesis alternativa (H1), ya que el 0 queda fuera del intervalo de confianza, indicando que la diferencia es estadísticamente significativa.

**Extra**

Ключевое правило: ДИ для разности средних содержит 0 → нет значимой разницы (H0 не отвергается). ДИ не содержит 0 → разница значима (H0 отвергается). Аналогия с p-value: исключение 0 из ДИ эквивалентно p < α. Слово «acepta» берётся в кавычки, так как строго говоря мы не доказываем H1, а лишь не можем сохранить H0.
