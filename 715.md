## Core

Odds Ratio считается статистически значимым, когда его доверительный интервал НЕ включает нулевое значение (1 для OR).

## Final Decision

Два unit-а объединены в одну карточку: вопрос спрашивает условие значимости OR, ответ включает ключевой факт, что нулевое значение для OR = 1. Оба unit-а критичны, но естественно формируют один memory failure point.

**Covers**

- OR статистически значим, когда ДИ не включает нулевое значение (1)
- Нулевое значение для OR равно 1 (а не 0)

**Front**

¿Cuándo se considera que una Odds Ratio (OR) tiene resultado estadísticamente significativo?

**Back**

Cuando el intervalo de confianza (IC) NO incluye el valor nulo, que para la OR es 1.

**Extra**

Ключевая ловушка: нулевое значение для OR — это 1, а не 0. Если ДИ включает 1 (например, IC 95%: 0,8–1,5), OR не значима. Если ДИ не включает 1 (например, IC 95%: 1,2–3,4), OR статистически значима. Для разницы рисков (Risk Difference) нулевое значение = 0, для относительного риска (RR) — тоже 1.
