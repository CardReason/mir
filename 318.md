**Covers**

- Если p значимо → ДИ для разности пропорций исключает нулевое значение (0)
- Нулевое значение для разности пропорций = 0

**Front**

¿Qué condición debe cumplir el intervalo de confianza para la diferencia de proporciones cuando el valor de "p" es estadísticamente significativo?

**Back**

El intervalo de confianza para la diferencia de proporciones debe excluir el valor nulo (0).

**Extra**

Правило соответствия: если p < 0,05 → ДИ для разности пропорций не содержит 0. Если ДИ содержит 0 (например, от −0,05 до +0,10), разность может быть нулевой → результат статистически незначим. Нулевое значение для разности пропорций всегда равно 0 (в отличие от относительного риска/ОШ, где нулевое значение = 1).
