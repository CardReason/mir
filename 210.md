## Core

Различие между двумя формами фентанила: сублингвальный — для прорывной боли, трансдермальный — для фоновой боли.

## Final Decision

Один contrast-карточка покрывает оба юнита — сублингвальный vs трансдермальный. Второй юнит встроен как пара. Бюджет не превышен.

**Front**

¿Qué vía de administración del fentanilo se usa para el dolor irruptivo y cuál para el dolor de fondo?

**Back**

Dolor irruptivo → fentanilo sublingual
Dolor de fondo → fentanilo transdérmico

**Extra**

Ключевая логика: трансдермальный пластырь обеспечивает медленное и стабильное высвобождение → подходит для постоянной фоновой боли. Сублингвальный фентанил всасывается быстро → подходит для острых эпизодов прорывной боли на фоне базовой анальгезии.
