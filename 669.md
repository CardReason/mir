**Covers**

- Wilcoxon compara medias (medianas), no proporciones
- Para comparar proporciones → Chi-cuadrado

**Front**

¿Qué prueba estadística se usa para comparar medias (medianas) y cuál para comparar proporciones?

**Back**

Comparar medias/medianas → Wilcoxon (test no paramétrico)
Comparar proporciones → Chi-cuadrado (χ²)

**Extra**

Частая ошибка: применять Уилкоксон для сравнения пропорций. Уилкоксон — непараметрическая альтернатива t-критерию, работает с порядковыми данными и медианами. Хи-квадрат — для категориальных данных и пропорций (например, частота события в двух группах).
