**Front**

¿Cómo se define y calcula la Sensibilidad de una prueba diagnóstica?

**Back**

Capacidad de detectar enfermos.
Fórmula: Sensibilidad = VP / (VP + FN)
(Verdaderos Positivos / total de enfermos reales)

**Extra**

Чувствительность показывает, какую долю реально больных тест правильно идентифицирует как больных. VP = verdaderos positivos; FN = falsos negativos. Высокая чувствительность → мало ложноотрицательных → тест хорош для скрининга (не пропускает больных).
