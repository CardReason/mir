**Covers**

- Хи-квадрат используется для сравнения пропорций
- Применяется к качественным (категориальным) переменным

**Front**

¿Para qué se utiliza la prueba Chi-cuadrado y a qué tipo de variables se aplica?

**Back**

Compara proporciones entre dos o más variables cualitativas (categóricas).

**Extra**

Ключевое правило: хи-квадрат применяется только к качественным переменным. Для количественных переменных используют другие тесты (t-Студента, ANOVA). На экзамене часто спрашивают: какой тест выбрать — смотри на тип переменной: если категориальная → хи-квадрат.
