ANOVA — статистический метод для сравнения средних значений более чем в двух группах

**Covers**

- ANOVA se usa para comparar medias en más de dos grupos

**Front**

¿Qué test estadístico se utiliza para comparar medias de una variable cuantitativa entre más de dos grupos independientes?

**Back**

ANOVA (Análisis de la Varianza)

**Extra**

ANOVA применяется именно когда групп >2. Для двух групп используется t-Student. Типичная ошибка на экзамене — применить t-Student к трём и более группам.

**Covers**

- Variable dependiente cuantitativa, variable independiente categórica con >2 grupos
- Alternativa no paramétrica: Kruskal-Wallis

**Front**

¿Qué tipo de variables requiere el ANOVA y cuál es su alternativa no paramétrica?

**Back**

Variable dependiente: cuantitativa (continua). Variable independiente: categórica con más de 2 grupos. Alternativa no paramétrica: test de Kruskal-Wallis.

**Extra**

ANOVA предполагает нормальное распределение и гомогенность дисперсий. Если эти условия не выполнены — применяется непараметрический аналог Kruskal-Wallis, сравнивающий медианы вместо средних.
